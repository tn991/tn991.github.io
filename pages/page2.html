CVE-2020-35775(LDAP Injection)
CITSmart before 9.1.2.23. February 15, 2021;
CVE-2021-3331(WinSCP before 5.17.10, remote code execution)
allows remote attackers to execute arbitrary programs when the URL handler encounters a crafted URL that loads session settings. (For example, this is exploitable in a default installation in which WinSCP is the handler for sftp:// URLs.) January 27, 2021
CVE-2020-28472(aws-sdk before 2.814.0)
the package @aws-sdk/shared-ini-file-loader before 1.0.0-rc.9; the package aws-sdk before 2.814.0. If an attacker submits a malicious INI file to an application that parses it with loadSharedConfigFiles , they will pollute the prototype on the application. January 19, 2021
CVE-2020-29600(AWStats through 7.7)
cgi-bin/awstats.pl?config= accepts an absolute pathname, even though it was intended to only read a file in the /etc/awstats/awstats.conf format. December 07, 2020
reference: https://nvd.nist.gov/vuln/search/results?form_type=Basic&results_type=overview&query=aws&search_type=all
