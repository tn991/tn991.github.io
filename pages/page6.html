New Feature - EFS One-Zone Storage Class

... Cheaper option for using EFS is available now.

EFS, Elastic File System, is a service in AWS that provides a file system for both on-premise and cloud instances. Unlike EBS, Elastic Block Store, EFS is capable of being accessed by multiple instances simaltanesly therefore it enables users to have shared file system for various use cases.

The new feature, One-Zone Storage Class, means the data is stored in only one availability zone. Although the use of this feature reduces the fault-tolerance for availability zone failure, it provides cheaper option for users with less focus on fault tolerence and more focus on cost performance.
